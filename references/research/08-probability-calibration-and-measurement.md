# Probability, Calibration, And Measurement

## Core Figures

Thomas Bayes, Daniel Kahneman, Charles Goodhart, John Tukey

## Why This File Exists

Ground uncertainty handling in base rates, forecast humility, calibrated ranges, and metrics that resist gaming.

## Source Anchors

- Bayesian and base-rate discipline
- Kahneman on inside-view error and overconfidence
- Goodhart on target distortion and proxy failure
- Tukey on approximate useful measures over false precision

## Distinctions That Matter

- range versus false point precision
- signal versus proxy
- measurement for learning versus measurement for performance theater
- forecast confidence versus forecast calibration

## Canonical Questions

- What base rates or outside-view anchors belong here?
- What should the uncertainty be expressed as: point, range, or scenario set?
- Which metric is genuinely informative and which is merely easy to steer?
- What guardrails keep the lead metric from corrupting behavior?

## What This Changes In The Skill

- It disciplines forecasts and confidence claims with base rates, calibration, and measurement quality.
- It helps the skill resist false precision and metric vanity.
- It improves how the skill moves from narrative to evidence-backed uncertainty handling.

## Good Moves

- replace certainty language with calibrated ranges and conditions
- design metric sets with a lead signal and explicit guardrails
- measure what helps a decision, not what flatters a dashboard
- prefer simple useful measures to elaborate but gameable ones

## Common Misuse

- pretending unknowns are smaller than they are
- using one metric to stand in for the whole system
- mistaking internal dashboard visibility for genuine forecast skill
- rewarding targets in ways that distort the underlying behavior

## Example Translation

“When will AGI arrive?” should not start with a year guess. It should start with dimensions, base rates, error bars, and the evidence that would move the forecast materially.

Use this file when the answer depends on uncertainty, forecasting, metrics, evidence quality, or the risk of optimizing the wrong proxy.
