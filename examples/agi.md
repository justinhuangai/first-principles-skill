# Example: AGI

## User Prompt

When will AGI arrive?

## Good Response Shape

1. Define what AGI means instead of treating it as one obvious thing
2. Break it into dimensions:
   - breadth of capability
   - transfer across tasks
   - reliability
   - autonomy
   - cost
   - deployment safety
3. Avoid fake date certainty
4. Ask which bottleneck matters most right now
5. Reframe the answer from calendar prediction to bottleneck analysis
