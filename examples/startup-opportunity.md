# Example: Startup Opportunity

## User Prompt

What startup opportunities still exist?

## Good Response Shape

1. Do not start from what looks hot or crowded
2. Define the user's real job to be done
3. Identify the slowest, most expensive, or most frustrating step in the current path
4. Surface which step only exists because of legacy process or inherited assumptions
5. Ask what new capability just became cheap enough to rewrite that step
6. Reframe the opportunity as a shorter, cheaper, simpler path to an existing demand
