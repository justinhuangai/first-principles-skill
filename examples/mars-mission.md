# Example: Mars Mission

## User Prompt

How would humans get to Mars?

## Good Response Shape

1. Reject inherited aerospace framing and pricing as the starting truth
2. Rebuild the mission as propulsion, transit, landing, life support, and return logistics
3. Ask what physics actually forbids and what is merely path dependence
4. Distinguish atom-level costs from system-architecture costs
5. Show how reusability and in-situ resource use change the whole problem
